# 🚀 RL-Based Bin Picking Robotics

## 📌 Overview

This project focuses on solving robotic bin-picking using Reinforcement Learning (PPO/SAC).

## 🧠 Tech Stack

* Python
* PyBullet
* RoboDK
* Reinforcement Learning (PPO, SAC)

## ⚙️ System Workflow

1. Environment simulation using PyBullet
2. Robot motion via RoboDK
3. RL agent learns optimal picking path

## 📊 Results

* Stable convergence after training
* Improved picking efficiency
